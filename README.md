# Excel Pivot Table Analysis

## Project Overview
This repository contains an analysis of the Sample Superstore dataset using Excel pivot tables, charts, and slicers. 
The goal is to derive business insights on sales performance, category trends, and regional performance.

## Files in this Repository
- `Pivot_Report.xlsx` - Excel file with pivot tables, slicers, and charts
- `Pivot_Report.pdf` - PDF version of Excel pivot tables
- `Sales_Report.pdf` - Word report with analysis, insights, and screenshots

## Dataset
- Sample Superstore (Kaggle)
- ~10,000 rows
- Columns: Order Date, Category, Sub-Category, Region, Segment, Sales, Profit, Quantity, Discount

## Key Insights
1. West region has the highest sales.
2. Technology category performs best.
3. Consumer segment dominates total sales.
4. High discounts reduce profit in some regions.
5. South region requires focused attention.
